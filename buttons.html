Buttons

Reusable button styles and interaction states for the LifeNote interface.

Components

Buttons Cards Inputs Navigation

Overview

Variants

States

Purpose

Buttons support major user actions across the app, including journal creation, navigation, and feature access.

Usage

  • Use primary buttons for main actions.
  • Use secondary buttons for supporting actions.
  • Use accent buttons sparingly for emphasis.
  • Keep labels short and action-focused.
