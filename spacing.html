Spacing

Spacing guidelines for layout rhythm, card padding, section gaps, and consistent alignment across LifeNote.

Foundations

Overview

Spacing Scale

  • 8px: Small gaps
  • 12px: Tight spacing
  • 16px: Default spacing
  • 24px: Card padding
  • 32px: Section spacing
  • 48px: Large layout spacing
  • 64px: Major page sections

Best Practices

  • Use spacing tokens instead of random values.
  • Keep card padding consistent.
  • Use larger spacing between major sections.
  • Use smaller spacing between related elements.

Purpose

Applied in Interface
