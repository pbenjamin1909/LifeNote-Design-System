Icons

Icon guidelines for simple, consistent visual cues across LifeNote’s journaling, navigation, and insight features.

Foundations

Color Typography Spacing Icons

Overview

Icon Style

  • Use simple outline-style icons.
  • Keep icons clear and minimal.
  • Use icons to support text, not replace it.
  • Use consistent sizing across the interface.

Best Practices

  • Pair icons with labels when meaning may not be obvious.
  • Use the primary color for active icons.
  • Use muted color for inactive icons.
  • Avoid mixing filled and outline styles.

Purpose

Icons help users scan actions and sections quickly while supporting the written label.

Accessibility

Icons should not be the only way meaning is communicated. Use text labels when possible.
