Overview

Color Roles

The LifeNote color system uses a small, defined palette to create clear visual hierarchy, support readability, and keep actions consistent across the interface.

  • Primary: Main actions
  • Primary Hover: Interaction feedback
  • Accent: Emphasis
  • Background: Page layout
  • Surface: Cards and containers
  • Text Primary: Main readable text
  • Text Secondary: Supporting text
  • Border: Dividers and outlines

Best Practices

  • Use primary blue for the main action in a section.
  • Use accent purple sparingly for emphasis.
  • Maintain strong contrast between text and background.
  • Use surface and background colors consistently.
  • Avoid introducing colors outside the system.
